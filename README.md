# url-resemblance README

Features

> Identifies url resemblance and throws errors with corresponding line number.

## Requirements

Requires Python3.

VSC version ^1.22.2.

## Extension Settings

No special settings required.

## Known Issues

Supports only HTTP Go routes, with file routes file name as **routes.go**.
